# 📚 LitRate

**LitRate** — это веб-приложение для оценки книг и написания рецензий, разработанное в рамках семестрового проекта студентами:

- 🧑‍💻 **Кирсанов Егор**  
- 👨‍💻 **Лескин Евгений**  
- 👩‍💻 **Коряева Полина**  
- 🧑‍💻 **Косарев Дмитрий**

---

## ✨ О проекте

LitRate позволяет пользователям:

- 📖 Кратко знакомиться с содержанием книг  
- ⭐ Оценивать прочитанные произведения  
- ✍️ Писать и публиковать рецензии  
- 👥 Подписываться на других пользователей  
- 📚 Просматривать чужие рецензии и книжные полки  

Проект создан для удобного взаимодействия читателей и объединения их в книжное сообщество.
